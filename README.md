# BMI Calculator (Java)

## üìå Description

This is a simple **Java console application** that calculates a person's **Body Mass Index (BMI)** using their **height (in inches)** and **weight (in pounds)**. Based on the calculated BMI value, the program classifies the person as **Underweight, Healthy weight, Overweight, or Obese**.

---

## üìÇ Package Information

* **Package Name:** `day3`
* **Class Name:** `BMICal`

---

## üßÆ Formula Used

The BMI is calculated using the standard formula for pounds and inches:

```
BMI = (weight / (height * height)) * 703
```

---

## üñ•Ô∏è How the Program Works

1. The program asks the user to enter their **height in inches**.
2. Then it asks for **weight in pounds**.
3. It calculates BMI using the formula.
4. Based on the BMI value, it prints the health category:

   * **Underweight**: BMI < 18.5
   * **Healthy weight**: 18.5 ‚Äì 24.9
   * **Overweight**: 25 ‚Äì 29.9
   * **Obese**: 30 and above
